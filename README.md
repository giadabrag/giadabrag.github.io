# GiadaBrag.github.io

Indice semplice per le app pubblicate su GitHub Pages.

## Come aggiungere nuove categorie o app
- Crea la cartella dell'app (es. `scienze/esperimento1/`)
- Aggiungi la voce in `data/apps.js`

La pagina principale (`index.html`) legge i dati da `data/apps.js` e genera
automaticamente le card.
